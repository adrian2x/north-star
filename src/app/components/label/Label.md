```jsx
<div className="column">
  <Label className="mr-1">Default</Label>
  <Label className="mr-1" variant="primary">
    Primary
  </Label>
  <Label className="mr-1" variant="secondary">
    Secondary
  </Label>
  <Label className="mr-1" state="success">
    Success!
  </Label>
  <Label className="mr-1" state="warning">
    Warning...
  </Label>
  <Label className="mr-1" state="error">
    Error!
  </Label>
</div>
```

### Rounded labels

Set the `rounded` prop to have rounded labels.

```jsx
<div className="column">
  <Label className="mr-1" rounded>
    Default
  </Label>
  <Label className="mr-1" variant="primary" rounded>
    Primary
  </Label>
</div>
```
